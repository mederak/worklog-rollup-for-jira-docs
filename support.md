## How To Report A Problem

Send a support request with a clear description of the problem and the steps needed to reproduce it.

Please include:

- Jira Cloud site URL, without sensitive path details if possible,
- app version,
- description of the problem,
- screenshot with sensitive data blurred or removed,
- issue type where the problem appears,
- whether the problem affects one issue or multiple issues,
- whether the issue has child issues or a deeper hierarchy.

## What Not To Send

Do not send:

- customer data that is not needed for troubleshooting,
- confidential business information,
- secrets,
- API tokens,
- passwords,
- full personal data,
- unredacted screenshots containing sensitive content.

## Support Scope

Support covers problems related to the app's installation, display, calculation behavior, and documented features.

Support does not cover:

- Jira administration unrelated to the app,
- correcting Jira worklog or time tracking data,
- third-party reporting tools,
- custom billing or payroll workflows.

## Response Time

Support is provided on a best-effort basis.

Typical response time is 2-5 business days.

24/7 support is not guaranteed.
